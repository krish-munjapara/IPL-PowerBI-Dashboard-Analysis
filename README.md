# IPL Analysis Dashboard (2008–2025) | Power BI

## Project Overview
This project analyzes Indian Premier League (IPL) data from 2008 to 2025 using Power BI.
It provides interactive dashboards to visualize team performance, player statistics, and season-wise insights.

## Tools Used
- Power BI
- Power Query
- DAX
- CSV datasets

## Key Features
- Season-wise slicer (2008–2025)
- Champion and Runner-up highlights
- Orange Cap and Purple Cap statistics
- Points table with team performance
- KPI cards (Matches, 4s, 6s, Centuries)

## How to Use
1. Download the PBIX file
2. Open it in Power BI Desktop
3. Select season to view insights
